Configuring an autonomous workstation on your PC

Warning: for autonomous work without connection to the Internet, you need to preliminarily purchase and install the [НазваниеПрограммы] application locally on your computer. For more details, see below.

  1. Create a new folder where your standalone workstation infobase will be stored. We recommend that you create this folder in the "My documents" folder.
  2. Open the downloaded archive file Standalone mode.zip and extract all files from it to the folder you created.
  3. Start 1C:Enterprise. In the "1C:Enterprise startup" window, click "Add…"
  4. Select Add infobase and click Next to proceed to the next wizard page.
  5. Select This computer or LAN computer and type in the infobase name. We recommend that you use the same name for your standalone workstation and your infobase.
  6. Specify Infobase subfolder located in the folder you created during step 1, and click Finish.
  7. Click 1C:Enterprise (located at top right of the 1C:Enterprise startup window) to start 1C:Enterprise.
    In infobase access window, select the creator of the standalone workstation from the list in the User field. The Password field must remain blank.

 

To start working with the application, you will always need to start 1C:Enterprise and select this infobase from list in the 1C:Enterprise startup window. To modify the Internet data synchronization parameters, use the Standalone mode command in the Administration section.

Additional information

 

1. How to install or update the 1C:Enterprise platform version?

Important: if you want to work in a standalone mode (without an Internet connection), you will need to purchase and install [НазваниеПрограммы] on your computer. 1C:Enterprise applications are available through a worldwide partner network.

If the application is already installed on your computer, make sure that your 1C:Enterprise platform version is [ВерсияПлатформы] or higher (to look up the platform version, check information in the About window available via Help - About menu command). If your 1C:Enterprise platform version is outdated, you will need to download and install the latest version from 1C:Enterprise user support website.

2. How to set up the autonomous workplace for work of other users?

As soon as you have completed the setup procedure above, your standalone workstation will be ready for work. Your account will be granted the administrative rights, identical to the Internet-enabled mode.

To share this standalone workstation with other users, you need to allow these users to sign in to the application. For this, add them to the Users list located in the Administration section. This will grant them access rights identical to the Internet-enabled mode. For more information on user list maintenance and access rights setup, refer to the application documentation.

To allow the users to perform Internet data synchronization without requiring your assistance, you will additionally need to modify the default Internet connection settings. For this, you may use the button "Set up connection..." in the window "Autonomous work" of the section "Administration". In the setup form, specify a user to perform data synchronization. Assign Data synchronization with other programs profile for this user in the Internet-based version of the application.
