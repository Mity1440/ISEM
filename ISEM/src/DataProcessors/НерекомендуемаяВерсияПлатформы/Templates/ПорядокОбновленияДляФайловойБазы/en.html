Version update procedure for 1C:Enterprise 8.3

1. Download the 1C:Enterprise 8.3 distribution package

In 1C:Application update service, download the latest version of the 1C:Enterprise 8.3 platform distribution package. If required, type username and password.

Select the latest officially issued version of the Technological platform 8.3.

To download a file, click a hyperlink that matches your operating system:

  • 1C:Enterprise technological platform for Windows
  • 1C:Enterprise technological platform for Linux
  • 1C:Enterprise client for macOS

Click the Download distribution package link and set the file path. Remember the folder the file is saved to.

2. Install the new version of 1C:Enterprise 8.3

If you use Microsoft Windows OS, you must have administrator rights to install the program. Extract the archive. In the folder with the extracted archive, run the Setup.exe file and follow the installation wizard instructions.

3. Create the backup infobase copy

Run 1C:Enterprise. In the 1C:Enterprise startup window, define and open a folder with the required infobase. Copy the 1Cv8.1CD file to a safe place.

4. Start to work with the new version of the platform

Click the 1C:Enterprise to start working with the selected infobase on the new platform version.
